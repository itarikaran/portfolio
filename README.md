# Ari Karan - Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and experience as a Full-Stack Developer. Built with React, Vite, and Tailwind CSS with smooth animations and a professional design.

## 🚀 Live Demo

[Visit Portfolio](https://arikaran.dev) *(Update with your live URL)*

## ✨ Features

- **Responsive Design** - Optimized for mobile, tablet, and desktop
- **Smooth Animations** - Framer Motion animations for engaging interactions
- **Dark Theme** - Eye-friendly dark mode design with red accent colors
- **SEO Optimized** - Meta tags and structured data for better search visibility
- **Accessible** - WCAG 2.1 compliant with proper ARIA labels
- **Fast Performance** - Code splitting, lazy loading, and optimized bundle size
- **Error Handling** - Global error boundary for graceful error display
- **TypeScript Ready** - Full type safety support

## 🛠️ Tech Stack

- **Frontend**: React 19 + Vite 8
- **Language**: TypeScript (optional)
- **Styling**: Tailwind CSS + PostCSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Linting**: ESLint

## 📋 Sections

- **Hero** - Introduction with CTA buttons
- **Stats** - Quick statistics and achievements
- **About** - Professional summary
- **Skills** - Technical skills breakdown
- **Learning** - Continuous learning initiatives
- **Projects** - Showcase of completed projects
- **Experience** - Professional work experience
- **Education** - Academic background
- **Certifications** - Relevant certifications
- **Profiles** - Social and professional profiles
- **Contact** - Contact information and form

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/itarikaran/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Development

```bash
# Start Vite dev server with HMR
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview

# Run ESLint
npm run lint
```

## 🎨 Customization

### Update Personal Information

Edit the content in these components:
- `src/components/Hero.jsx` - Update name and headline
- `src/components/About.jsx` - Update bio
- `src/components/Contact.jsx` - Update contact details
- `src/components/Projects.jsx` - Add your projects

### Modify Colors

Edit `tailwind.config.js` to change the accent color scheme.

### Add Resume

Place your resume PDF at `public/resume.pdf`
